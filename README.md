# Amazon Monthly Sales Dashboard Analysis

## 📌 Project Overview
This project delivers an interactive Data Analytics Dashboard designed to monitor, analyze, and optimize Amazon monthly sales performance. It synthesizes critical retail metrics across sales, order volumes, product categories, geographic distribution, and fulfillment efficiency to drive data-backed decision-making.

## 📊 Key Performance Indicators (KPIs)
The dashboard tracks three primary high-level business metrics:
* **Total Sales:** ₹76.03M gross revenue across the monitored period.
* **Quantity Sold:** 117K individual product units ordered.
* **Average Order Value (AOV):** ₹648.56 generated per transaction.

## 🗺️ Dashboard Architecture & Visualizations
The layout is optimized for executive scanning and detailed root-cause analysis:
1. **Temporal Filters:** Interactive month-wise selectors (March, April, May, June).
2. **Geographic Distribution:** A dynamic symbol map plotting state-wise sales distribution across India.
3. **Product Segmentation:** 
   * A donut chart illustrating *Category Wise Sales* revenue share.
   * A bar chart displaying *Category Wise Customer* volume.
4. **Logistics Performance:** A pie chart detailing *Order Wise Courier Status* (Shipped, Unshipped, Cancelled).

## 🚀 Key Insights Preview
* **Core Revenue Drivers:** Traditional attire ("Set" and "Kurta") dominates both unit volume and total revenue share.
* **Fulfillment Efficiency:** Exceptional logistics health with the vast majority of orders successfully transition to "Shipped" status.
* **Regional Hubs:** High concentration of sales volume originating from central, western, and southern peninsular states of India.

## 🛠️ Tech Stack & Tools
* **Data Visualization:** Microsoft Power BI .
* **Data Sources:** Amazon Seller Central transactional exports.
